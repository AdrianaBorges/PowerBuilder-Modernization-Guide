# Guia de Modernização do PowerBuilder

Este repositório reúne conceitos, experiências práticas e estratégias utilizadas na modernização de aplicações PowerBuilder em ambientes corporativos.

O objetivo é compartilhar conhecimento sobre manutenção, evolução e integração de sistemas legados, preservando regras de negócio críticas enquanto se viabiliza a adoção de arquiteturas modernas.

---

## Temas abordados

### PowerBuilder

- Arquitetura Cliente/Servidor
- Ciclo de vida das aplicações
- Boas práticas de desenvolvimento
- Performance e manutenção

### DataWindows

- Estrutura e funcionamento
- Performance
- Problemas comuns
- Estratégias de manutenção

### PFC (PowerBuilder Foundation Classes)

- Conceitos fundamentais
- Benefícios
- Padrões utilizados
- Boas práticas

### Bancos de Dados

- Sybase SQL Anywhere
- SQL Server
- PostgreSQL
- Procedures
- Triggers
- Views
- Otimização de consultas

### Modernização de Sistemas

- Migração PowerBuilder 16 para 32 bits
- Integração PowerBuilder e .NET
- Integração com APIs REST
- Estratégias de modernização gradual
- Redução de riscos em projetos legados

### Integração

- Web Services
- APIs REST
- XML
- JSON
- Middleware
- Arquitetura de integração
